# AudMX

Alternative for Deej ( https://github.com/omriharel/deej ) with more features

- Deej compatible
- RGB
- Looks cool
- Bluetooth
- Compact

Must be used with PC app for full features access ( https://github.com/Bobla-2/AudMx-win )
